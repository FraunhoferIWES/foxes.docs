    foxes.models.wake_models.wind.porte_agel.PorteAgelModel¶

    class foxes.models.wake_models.wind.porte_agel.PorteAgelModel[source]¶

    Bases: object

    Common calculations for the wake model and the wake frame, such that code repetitions can be avoided.

    Based on Bastankhah & Porte-Agel, 2016, https://doi.org/10.1017/jfm.2016.595

    Parameters:¶
    ct_max : float

    The maximal value for ct, values beyond will be limited to this number, by default 0.9999

    alpha : float

    model parameter used to determine onset of far wake region

    beta : float

    model parameter used to determine onset of far wake region
