    foxes.input.states.create.random_abl_states.create_random_abl_states¶

    foxes.input.states.create.random_abl_states.create_random_abl_states(n_states, cols_minmax, var2col={}, mol_abs_range=(50.0, 5000.0), normalize=True)[source]¶

    Create random abl states.

    Parameters:¶
    n_states : int¶

    The number of states

    cols_minmax : dict¶

    For each variable the min and max values, keys: variable name str, values: array_like with length 2

    var2col : dict, optional¶

    Mapping from variables to column names

    mol_abs_range : tuple¶

    Min and max of allowed MOL values, set to nan if exceeded (i.e., neutral stratification)

    normalize : bool¶

    Normalize weights to 1

    Returns:¶

    data – The created states data

    Return type:¶

    pandas.DataFrame
