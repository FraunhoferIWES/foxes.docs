    Executes strings that start with the indicator as python commands, returns one value

    Example: s = “%p%N=10;p=np.zeros((N,3));p[:,0]=50;p[:,1]=np.linspace(0,7000,N);p[:,2]=119”

    Parameters¶

    s: list, dict or object

    The source to by analyzed

    indicator: str

    The indicator that trigger python evaluation

    newline: str

    The new line indicator

    globals: dict

    The global namespace

    locals: dict

    The local namespace

    Returns¶

    out: list, dict or object

    The same structure, but all python strings evaluated
